SwiftUI-认识基本界面

1.主区域

1.1 Navigator Area/导航区域

导航区域用于查找资源、定位书签、排错…

1.2 Debug Area/调试区域

代码区域与编辑区域共称调试区域

代码区域用于编辑项目代码

编辑区域跟随代码区域变动

1.3 Inspector Area/检查区域

检查区域用于显示当前资源属性、控件属性…

当代码区域定位到具体语法时，在检查区域可以视图方式调整值，随后编辑区域、代码区域被动同步

2.详细分区

2.1 Navigator/导航

2.1.1 Project Navigator/项目导航

项目导航用于资源查找

2.1.2 Source Control Navigator/资源控制导航

资源控制导航用于观察Git仓库变化及控制

2.1.3 Bookmark Navigator/书签导航

书签导航用于快速定位代码区域中的行书签

2.1.4 Find Navigator/查找导航

查找导航用于快速查找代码

2.1.5 Issue Navigator/错误导航

错误导航用于排错

2.1.6 Test Navigator/测试导航

测试导航用于测试代码API调用、生存周期…

2.1.7 Debug Navigator/调试导航

调试导航用于记录调试会话

2.1.8 Breakpoints Navigator/断点导航

断点导航用于对代码进行局部调试

2.1.9 Report Navigator/报告导航

报告导航用于记录构建会话，若构建失败后方会出现错误标识

2.2 Inspector/检查器

2.2.1 File Inspector/文件检查器

文件检查器用于显示当前编辑文件的名称、种类、路径…

2.2.2 History Inspector/历史检查器

历史检查器用于复盘之前的检查

2.2.3 Quick Help Inspector/快速帮助检查器

快速帮助检查器用于提示当前控件的语法和用途

2.2.4 Accessibility Inspector/易用性检查器

易用性检查器用于测试一些辅助功能如旁白

2.2.5 Attributes Inspector/属性检查器

属性检查器用于修改控件的属性，如文本控件的大小字体…

2.3 Toolbar/工具栏

红框部分共称工具栏

黄框部分控制导航区域和检查区域的显隐

绿框部分为构建按钮

蓝框部分为控件库，可拖拽至编辑区域

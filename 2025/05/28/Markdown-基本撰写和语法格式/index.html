2.2加粗&斜体

加粗

ABC比ABC要粗些

代码示例
**ABC**

斜体

ABC是ABC的斜体

代码示例
*ABC*

Note
全部加粗及为斜体

ABC不仅加粗还是斜体

代码示例
***ABC***

全部加粗部分斜体

ABCDEF全部加粗但CD为斜体

代码示例
**AB*CD*EF**

全部斜体部分加粗

ABCDEF全部为斜体但CD加粗

代码示例
*AB**CD**EF*

2.3上标&下标

上标

ABCDEF中的CD更高

代码示例
AB<sup>CD</sup>EF

下标

ABCDEF中的CD更低

代码示例
AB<sub>CD</sub>EF

2.4删除线&下划线

删除线

我喜欢她但我更喜欢无畏的自己

代码示例
~~我喜欢她~~但我更喜欢无畏的自己

下划线

这很重要，划重点

代码示例
<ins>这很重要，划重点</ins>

3.文本颜色

3.1前景色

使用颜色名称格式

RED-红色

GREEN-绿色

BLUE-蓝色

代码示例
<font color="RED">RED-红色</font>
<font color="green">GREEN-绿色</font>
<font color="blue">BLUE-蓝色</font>

Note
使用HEX格式

RED-红色

GREEN-绿色

BLUE-蓝色

代码示例
<font color="#FF0000">RED-红色</font>
<font color="#008000">GREEN-绿色</font>
<font color="#0000FF">BLUE-蓝色</font>

3.2背景色

使用颜色名称格式

RED-红色

GREEN-绿色

BLUE-蓝色

代码示例
<font style="background: red">RED-红色</font>
<font style="background: green">GREEN-绿色</font>
<font style="background: blue">BLUE-蓝色</font>

Note
使用HEX格式

RED-红色

GREEN-绿色

BLUE-蓝色

代码示例
<font style="background: #FF0000">RED-红色</font>
<font style="background: #008000">GREEN-绿色</font>
<font style="background: #0000FF">BLUE-蓝色</font>

Tip
同时更改前景色与背景色
Live at ease like a carrot!

代码示例
<font style="background: #1F883D"><font color="#ffb000">Live at ease like a carrot!</font></font>

4.引用

4.1引用文本

右侧灰色索引及垂直线，字体为灰色

Calin的网名是隔壁萝卜

代码示例
>Calin的网名是隔壁萝卜

4.2引用代码

单行代码

sudo -i

代码示例
`sudo -i`

多行代码

sudo -i
apt update
apt upgrade
代码示例
```
sudo -i
apt update
apt upgrade
```

Note
代码中含有反引号

```
sudo -i
rm -rf /*
```
代码示例
#若即将引用代码带有反引号且数量>=3，则引用代码书写的反引号数量应+1
````
```
sudo -i
rm -rf /*
```
````

Tip
代码引用语法高亮

#无语法高亮示例
sudo -i
rm -rf /*
#有语法高亮示例
sudo -i
rm -rf /*
代码示例
#bash可为其他语言，如C,Java等
```bash
#有高亮语法示例
sudo -i
rm -rf /*
```

4.3引用头部（仅Github警告样式）

代码示例
> [!NOTE]
> Useful information that users should know, even when skimming content.

> [!TIP]
> Helpful advice for doing things better or more easily.

> [!IMPORTANT]
> Key information users need to know to achieve their goal.

> [!WARNING]
> Urgent info that needs immediate user attention to avoid problems.

> [!CAUTION]
> Advises about risks or negative outcomes of certain actions.

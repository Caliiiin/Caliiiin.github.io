2.2 注释

Markdown注释

用于在书写时在代码旁批注，最终不予显示

代码示例
<!-- 恨之入骨，爱之入髓，放下挂念，好好生活 -->

脚注（需渲染器支持）

代码示例
海内[^1]存知己，天涯[^2]若比邻[^3]

[^1]: 海内：四海之内，即全国各地，古代人认为我国疆土四周环海，所以称天下为四海之内。
[^2]: 天涯：天边，这里比喻极远的地方。
[^3]: 比邻：并邻，近邻。

3.2 标题/章节链接

跳转上一章节1.1图像

代码示例
跳转上一章节[4.1引用文本](#4-1引用文本)

Important
圆括号内不能包含空格、斜杠、点等，须用连字符代替
页内级别不同的标题内容一致时，圆括号内依旧保持一个井号
最小级可不用加连字符和数字，其上级需标记

代码示例
# How to be happy
## How to be happy
### How to be happy
跳转至[How to be happy](#-1How-to-be-happy)
跳转至[How to study well](#-2How to be happy)
跳转至[How to study well](#How to be happy)
